# ResumeAI - AI Powered Resume Analyzer

## Project Overview

ResumeAI is an AI-powered Resume Analyzer that compares a candidate's resume with a Job Description (JD) and generates an ATS compatibility score, recruiter insights, skill gap analysis, and personalized recommendations.

---

## Features

- Resume Upload (PDF/DOCX)
- Job Description Upload
- ATS Compatibility Score
- Skill Matching
- Missing Skills Detection
- AI Recommendations
- Recruiter Verdict
- Candidate Insights
- Interview Probability
- ATS Breakdown
- Skill Analytics Chart
- Job Fit Analysis
- Download Professional PDF Report
- Modern Responsive UI

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn api:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
